# Blog
Blog is project, that allow only admin users to add/update/delete pots and categories. and visitors to read posts. Implemented using Rails(ROR)

## Getting Started
**Usage**
```
#  cd Blog-master/
# rails s
```
 Open web browser
  ```
127.0.0.1:3000
  ```
